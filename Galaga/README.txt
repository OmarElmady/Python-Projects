README

To play Galaga 112, download the entire Galaga folder, and run the Galaga main file. Make sure that all the python files and the folders for music and images are in the same folder.

Controls:
Player 1:
Arrow keys to move, space to shoot
Player 2:
WASD to move, tab to shoot
At the end of the game, whoever has a higher score can input their name to the leaderboard!

Gameplay:
Shooting regular enemies grants 100 points; killing the boss in rage mode grants 500 points!
There are various powerups that grant improvements to your gun floating around in space; try to grab them to give yourself an edge against the enemies.
